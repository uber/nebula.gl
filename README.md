# Nebula.gl

A suite of 3D-enabled data editing overlays, suitable for deck.gl.



To run the example
-----
1. `git clone git@github.com:uber/nebula.gl.git`
2. `cd nebula.gl`
3. `yarn`
4. `cd examples/deck`
5. `yarn`
6. `yarn start-local`
7. You can view/edit geometry.
